# Analytical Thinking Driven by Design Thinking

## Product
E-Commerce Website

## User Research

### Target Users
Students, working professionals, and online shoppers.

### User Needs
- Easy product search
- Secure payment
- Fast delivery
- Reliable reviews

## User Persona

### Demographic Information
Name: Rahul Sharma
Age: 22
Occupation: Student

### Goals and Objectives
- Purchase products easily
- Save time and money

### Psychographic Information
- Technology enthusiast
- Prefers online shopping

### Behavior and Preferences
- Uses mobile apps frequently
- Compares products before buying

### User Journey
Awareness → Search → Compare → Cart → Checkout → Delivery

### Challenges and Pain Points
- Slow websites
- Delivery delays
- Fake reviews
